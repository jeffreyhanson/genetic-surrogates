DATA ORIGIN AND SAMPLING

This data package contains the genetic marker data (Amplified Fragment Length Polymorphisms) for 27 high-alpine species from the European Alps, as well as the geographic coordinates of every sampling location. The data belongs to the following paper:

Patrick G. Meirmans, Jérôme Goudet, IntraBioDiv Consortium, and Oscar E. Gaggiotti (2011). Ecology and life-history affect different aspects of the population structure of 27 high-alpine plants, Molecular Ecology.

The data has been collected as a part of the European research project IntraBioDiv, lead by Pierre Taberlet. For a complete list of members of the consortium see the included file "Consortium.pdf". The IntraBioDiv project was supported by the Commission of the European Union Sustainable Development, Global Change and Ecosystems Programme - GOCE-CT-2003-505376. We are grateful that the members of the IntraBioDiv consortium allowed us to use their data and make it available to the scientific community.

A detailed description of the sampling and genotyping protocols can be found in Alvarez et al. (2009, Ecology Letters, 12: 632-640), Thiel-Egenter et al. (2009, Global Ecology and Biogeography, 18: 78-87), and Gugerli et al. (2008, Perspectives in Plant Ecology, Evolution and Systematics, 10: 259-281). In short, sampling was performed across the European Alps on a grid with a cell size of 20’ longitude by 12’ latitude (approximately 20 x 22.5 km). Plant material was sampled from every second cell of the grid by taking leaves from three individuals at a single location within the cell, when a species was present in the cell. The number of locations sampled per species ranged from 44 to 137, and the maximum distance between sampling locations for the species ranged from 559 to 906 kilometers.

FILE DESCRIPTIONS
The dataset consists of 54 files, for every of the 27 species one file with the genetic data ("species_name_AFLP.dat"), and one file with the geographic coordinates ("species_name_locations.txt"). All files are in tab-delimited plain text format: UTF-8 with Unix line breaks.

GENETIC DATA FILES
The genetic data is stored in a haploid version of the Fstat file specification (Goudet 1995, Journal of Heredity, 86: 485-486). This format is defined as following:
-First line, four numbers:
         1) The number of populations,
         2) The number of loci,
         3) The maximum number used to code a allele (here 2)
         4) The number of digits used to code alleles (here 1)
         
-Next "number of loci"-lines: the names of the AFLP marker loci. These consist of the names of the selective primers used for the AFLPs followed by the location on the gel (for example: ACA_CAT_219.0)

-Next lines: For every individual a number indicating the population from which it was sampled, followed by for every locus the haploid AFLP genotype for that locus, coded as 1, 2, or 0:
         1: Absence of AFLP marker band
         2: Presence of AFLP marker band
         0: No data (NOT the same as band absence)
         
COORDINATE FILES
The coordinates of all sampling sites are also given in a separate file for every species. The files contain three columns of data:
-population: The name of the population given as the code of the sampled cell in the grid used by the IntraBioDiv project (see list below for all gridcells used). The populations are given in the same order as in the genetic data files.
-latitude: The latitude of the sampling location, given in decimal degrees
-longitude:  The longitude of the sampling location, given in decimal degrees
